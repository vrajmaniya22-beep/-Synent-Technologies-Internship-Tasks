## 🚀 Features

- ✅ **Fully Responsive** — Works on mobile, tablet, and desktop
- 🌙 **Dark Mode Design** — Deep navy/charcoal palette with purple accents
- ✨ **Smooth Animations** — Scroll-reveal, hover lifts, and micro-interactions
- 🎨 **Glassmorphism Navbar** — Frosted blur effect on scroll
- 🖼️ **Animated Hero Mockup** — Simulated design canvas with live multiplayer cursors
- ⚡ **Zero Dependencies** — Pure HTML, CSS & Vanilla JS — no frameworks

## 📁 Project Structure

```
lumina-landing/
├── index.html      # Main HTML structure
├── style.css       # All styles, responsive breakpoints & animations

```

## 🖥️ How to Run

```bash
# Clone the repository
git clone https://github.com/YOUR_USERNAME/lumina-landing.git

# Navigate to the project
cd lumina-landing

# Open in browser
open index.html
```

No build step needed. Just open `index.html` directly in any modern browser.

## 📱 Responsive Breakpoints

| Breakpoint | Layout |
|---|---|
| `> 900px` | Full desktop layout |
| `≤ 900px` | Single-column features, mobile nav |
| `≤ 600px` | Simplified hero, 2-col footer |
| `≤ 480px` | Fully stacked, single column |
