## 🚀 Features

- 🔄 **Seamless Toggling** — Smooth sliding transition between Login and Signup forms.
- 📐 **Dynamic Resizing** — The glassmorphism card smoothly animates its height to fit the active form.
- 🌙 **Aesthetic Dark Mode** — Clean minimal palette with purple glow accents and floating ambient orbs.
- 🔒 **Form Validation** — Real-time validation for email formatting, password length, and empty fields.
- 👁️ **Password Toggle** — Interactive eye icon to show/hide password visibility.
- 📱 **Fully Responsive** — Perfect rendering across mobile devices and desktop screens.
- 🌐 **Social Auth Grid** — Clean icon-only buttons for Google, Apple, Facebook, and GitHub.
- 🔔 **Toast Notifications** — Animated success messages upon form submission.

## 🖥️ How to Run

```bash
# Navigate to the task directory
cd "TASK 3"

# Open in your default browser
open index.html
```

No build step or dependencies required. Just open `index.html` directly in any modern browser.

## 🎨 Design System

| Token | Value | Use |
|---|---|---|
| **Primary Accent** | `#7c6aff` | Focus states, primary buttons, text links |
| **Background** | `#050508` | Deep space dark background |
| **Glass Surface**| `rgba(255,255,255,0.04)`| Main card background with backdrop-blur |
| **Text Primary** | `#eeeef4` | Headings and main body text |
| **Success Green**| `#22d3a5` | Ambient orb gradients & success toasts |
| **Error Red**    | `#ff6b6b` | Validation error borders and text |
